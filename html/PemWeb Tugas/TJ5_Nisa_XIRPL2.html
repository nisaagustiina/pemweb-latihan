Isi Artikel


  1. HTML

    2. Hypertext Markup Languange merupakan bahasa standar yang digunakan oleh browser internet untuk membuat halaman dan dokumen pada sebuah web yang kemudian dapat diakses dan dibaca seperti artikel.Selain itu, HTML juga digunakan sebagai link dalam situs.

  2. CSS

    3. Cascadung Style Sheet merupakan salah satu bahasa pemrograman web untuk mengendalikan beberapa komponen dalam sebuah web agar lebih terstruktur dan seragam.

  3. PHP

    4. Hypertext Preprocessor merupakan bahasa skrip yang ditanamkan ke dalam HTML.PHP banyak digunakan dalam web dinamis dan digunakan untuk membangun sebuah CMS.
